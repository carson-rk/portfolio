# Portfolio Website

This repository contains the source code for my personal portfolio website, showcasing one-page websites, landing pages, and small website fixes designed for small businesses.  

The website is **minimal, responsive, and mobile-first**, designed to highlight both design concepts (Figma) and clean front-end code (HTML, CSS, JavaScript).

---

## Features

- One-page portfolio layout  
- Mobile-friendly, responsive design  
- Sections for Services, Selected Work, Process, About, and Contact  
- Sample projects showcasing conceptual and real-world web solutions  
- Minimalistic design focused on clarity and usability

---

## Built With

- **HTML5** – Structure of the website  
- **CSS3** – Styling, layout, and responsiveness  
- **JavaScript** – Basic interactivity  
- **Figma** – Design mockups and prototypes
